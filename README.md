## Galvanize API assignment

## Running Locally

Create your `.env` file based on `.env.example`

First start:

- `yarn init:dev`

to setup docker/postgres

Then run:

- `yarn start`

To run the application with `ts-node`.


## Testing

You can run the example tests with:

- `yarn test`


## Build

Run:

- `yarn build` 

to compile from Typescript to Javascript
